## Usage 📦

- Clone or Downlaod this repo

```sh
git clone https://github.com/mskian/hello-world.git
cd hello-world
yarn install
```

- Test the Site

```sh
yarn develop
```

- Add post content - `/content/posts`

```sh
yarn newpost
```

- Production Build

```sh
yarn build
```

- Production Build store the Static Files on `dist` Folder
